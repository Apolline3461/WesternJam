# West

A small 2D point-and-click adventure made for a game jam themed **Western**.

---

## Context

This project is being developed as part of an online **game jam** hosted on itch.io.

The project is built entirely during the jam period and submitted as a playable game on itch.io, with a public GitHub repository showing the development progress.

---

## Project idea

My game is a short **2D point-and-click western adventure**.

You play as an outlaw trying to break into a bank vault before the sheriff returns.  
To succeed, the player must explore different locations in a small western town, collect useful items, interact with the environment, and solve simple puzzles.

## stack

- **Engine:** GDevelop
- **Version control:** Git / GitHub
- **Submission platform:** itch.io

The project uses GDevelop's **multiple files mode** so the development progress is easier to track in GitHub.

---

## Controls

- **Mouse**: interact with objects and scenes
- **Esc**: pause / back to menu
